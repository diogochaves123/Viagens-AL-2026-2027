# Pasta de Viagens — LEO Clube Soledade

Site institucional da Pasta de Viagens para apresentação do planejamento AL 2026/2027.

## Como rodar

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Imagens

Coloque os arquivos nas pastas abaixo:

- `public/images/logo-al-2026.png` — Logo oficial do AL (substitua o SVG temporário no cabeçalho)
- `public/images/diretores/diogo.png` — Foto do Diogo
- `public/images/diretores/gustavo.png` — Foto do Gustavo
- `public/images/diretores/bruno.png` — Foto do Bruno

Enquanto as fotos não forem adicionadas, o site exibe as iniciais de cada diretor.

## Build para produção

```bash
npm run build
npm run preview
```

Ou gere site estático:

```bash
npm run generate
```

## Tecnologias

- [Nuxt 4](https://nuxt.com)
- Vue 3
- Tailwind CSS
